# samples（フォーマット見本）

エージェントや会社から指定される **スキルシート / 職務経歴書のフォーマット見本** を入れるフォルダ。

## 入れ方

- 1フォーマット＝1ファイルで保存する（例：`agent-a.md`, `company-x.xlsx`, `keireki-template.docx`）
- ファイル名は提出先がわかる名前にする
- Excel/Word のテンプレートも可。構造の参考にする

## このフォルダの使われ方

`generate-skillsheet` skill が、ここにある見本のうち指定された1つの
**項目・章立て・順序・表記** を読み取り、`profile.md` の事実を当てはめて
`output/` に提出用ファイルを生成する。

> 見本に「本人が持っていない項目」があっても、事実を捏造して埋めない。
> 空欄または「（要確認）」にして本人に確認する。
